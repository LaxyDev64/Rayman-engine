COMPONENTES CLAVE
=================

1. MOTOR PRINCIPAL (engine.py)
   - GameEngine: Gestiona el bucle principal, escenas y control del juego

2. ENTIDADES (entities/)
   - Entity: Clase base abstracta para todas las entidades
   - Player: Jugador controlable (Rayman)
   - Platform: Plataformas estáticas
   - Wall: Paredes de colisión
   - Spike: Púas letales
   - Collectible: Objetos coleccionables

3. ESCENAS (scenes/)
   - Scene: Clase base abstracta para escenas
   - LevelScene: Escena de nivel con mecánicas de juego

4. SISTEMAS (utils/)
   - CollisionManager: Gestiona colisiones AABB
   - Animation: Maneja animaciones de sprites
   - AnimationController: Controla múltiples animaciones

5. CONFIGURACIÓN (config.py)
   - Parámetros de pantalla
   - Constantes de física
   - Configuración de controles
   - Colores y tamaños


FLUJO DE EJECUCIÓN
==================

1. main.py (inicio)
   ↓
2. Crear GameEngine
   ↓
3. Crear y registrar escenas (LevelScene)
   ↓
4. Establecer escena inicial
   ↓
5. GameEngine.run() - Bucle principal:
   ├─ handle_events() → Escena maneja input
   ├─ update(dt) → Actualizar entidades
   └─ draw() → Renderizar escena
   ↓
6. pygame.quit() cuando el usuario cierre el juego


CÓMO EXTENDER
==============

CREAR NUEVO ENEMIGO:
   1. Heredar de Entity
   2. Implementar update() y draw()
   3. Añadir a la escena en LevelScene.setup()

CREAR NUEVO NIVEL:
   1. Heredar de Scene
   2. Implementar setup(), handle_input(), update(), draw()
   3. Registrar en GameEngine
   4. Cambiar escena con game.set_scene()

CREAR NUEVO ITEM:
   1. Heredar de Entity
   2. Crear lógica de colisión
   3. Manejar en Scene.update()

NOTAS IMPORTANTES
=================

- El sistema de colisiones es AABB (rectángulos)
- Las coordenadas Y aumentan hacia abajo
- Las entidades tienen prioridad de actualización
- Las colisiones se resuelven en update()
- El jugador puede hacer doble salto
- Las púas resetean al jugador
- Los coleccionables desaparecen al tocarlos
